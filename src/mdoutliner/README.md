# Markdown Outliner

This is sample markdown content and the outline is generated from it.

## Features

- You can use the outline on the left to jump to parts of the document
- Demonstrates use of the Qt library bindings for Zig
- There is an option to change the language of the menu options

## Exercises (for the adventurous)

- Add a new language to the options
- Extend the language support to include the window title, tooltips, and open file dialog
- Port another one of the examples to this repository structure
- Flip the splitter view between the text and the outline
- Change the language options menu to show an indicator for the current language
